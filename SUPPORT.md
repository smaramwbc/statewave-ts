Start with the documentation:

## 💬 Community

For questions and discussions:

- **[GitHub Discussions](https://github.com/smaramwbc/statewave/discussions)** - Ask questions, share ideas
- Questions labeled `question` in any repo

## 🐛 Bug Reports

Found a bug? Please file an issue:

1. Search existing issues first to avoid duplicates
2. Use the bug report template
3. Include reproduction steps and environment details

[Report a Bug →](https://github.com/smaramwbc/statewave/issues/new?template=bug_report.yml)

## ✨ Feature Requests

Have an idea for Statewave?

1. Check the [roadmap](https://github.com/smaramwbc/statewave-docs/blob/main/roadmap.md)
2. Search existing feature requests
3. Open a feature request with your use case

[Request a Feature →](https://github.com/smaramwbc/statewave/issues/new?template=feature_request.yml)

## 🔒 Security Issues

**Do NOT open public issues for security vulnerabilities.**

Please email [security@statewave.ai](mailto:security@statewave.ai) instead.

See our [Security Policy](./SECURITY.md) for details.
